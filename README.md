# Paint-for-Browser
A simple paint program made to run in browser.

# Tools/Options for customisation
- Thin pen
- Line
- Line width
- Rectangle
- Circle
- Fill/Outline
- Rubber (work-in-progress)
- Undo
- Default colors
- Transparency (work-in-progress)
- Custom color (takes rgb, rgba, hex or color name)
- Save as
